01

Coleta de Dados

O AutoFill Brasil não coleta, transmite, vende, aluga nem compartilha nenhum dado pessoal do usuário.

Todos os dados inseridos na extensão — como CPF, RG, CNH, PIS/PASEP, e-mail, telefone, endereço e demais informações pessoais — são armazenados exclusivamente no dispositivo local do próprio usuário, por meio da API chrome.storage.local do navegador Google Chrome.

✔ Zero coleta. Nenhuma informação inserida pelo usuário é enviada a servidores, bancos de dados ou qualquer infraestrutura externa.
02

Armazenamento

Os dados são salvos localmente no navegador do usuário através da API chrome.storage.local, que armazena as informações no próprio dispositivo, de forma isolada e segura.

Esses dados não trafegam pela internet em nenhuma circunstância. Não existe servidor, banco de dados remoto, nuvem ou qualquer infraestrutura externa envolvida no funcionamento desta extensão.

03

Uso dos Dados

Os dados armazenados localmente são utilizados com um único propósito: preencher campos de formulários em páginas web, exclusivamente mediante ação explícita e voluntária do usuário ao clicar no botão "Preencher Formulário Agora".

A extensão não executa nenhuma ação em segundo plano, não monitora a navegação do usuário e não acessa nenhuma informação fora do contexto do preenchimento solicitado.

04

Compartilhamento

Nenhum dado é compartilhado com terceiros, anunciantes, parceiros comerciais ou qualquer outra parte, sob nenhuma circunstância.

✔ Zero compartilhamento. Esta extensão não possui integrações com serviços de analytics, publicidade ou rastreamento de qualquer natureza.
05

Segurança

Por não haver transmissão de dados pela internet, o risco de interceptação externa é inexistente no contexto desta extensão. A segurança dos dados armazenados localmente depende das configurações de segurança do próprio dispositivo e perfil do usuário no Chrome.

Recomendamos que o usuário mantenha seu sistema operacional e navegador sempre atualizados para garantir a máxima segurança dos dados locais.

06

Exclusão dos Dados

O usuário tem total controle sobre seus dados e pode removê-los a qualquer momento de duas formas:

Desinstalando a extensão — todos os dados armazenados pelo chrome.storage.local são automaticamente apagados pelo Chrome.

Limpando os dados do Chrome — acessando as configurações do navegador e limpando os dados de extensões instaladas.

07

Menores de Idade

Esta extensão não é direcionada a menores de 13 anos. Como não há coleta de dados, não existe risco de coleta inadvertida de informações de menores. Ainda assim, recomendamos que o uso seja supervisionado por responsáveis.

08

Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir melhorias na extensão ou mudanças em requisitos legais. A data da última atualização será sempre indicada no topo desta página.

Recomendamos que o usuário revise esta página ocasionalmente para se manter informado sobre eventuais mudanças.
